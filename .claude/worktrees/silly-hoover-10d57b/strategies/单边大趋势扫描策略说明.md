# 单边大趋势捕捉扫描策略 - 设计说明

## 📊 策略概述

这两个策略专门用于捕捉像 **RAVEUSDT** 这样在24小时内暴涨123.59%的单边大趋势行情。

- **基础版**: `单边大趋势捕捉扫描.py` - 使用24h基础数据进行快速扫描
- **高级版**: `单边大趋势捕捉扫描_高级版.py` - 使用多时间框架K线数据进行深度分析

---

## 🎯 核心设计思想

### 单边趋势的特征

通过观察 RAVEUSDT 的图表,我们发现单边大趋势具有以下特征:

1. **短期爆发性**: 价格在短时间内快速拉升,不回头
2. **成交量配合**: 上涨过程中成交量急剧放大
3. **持续性**: 连续多根K线同向运动
4. **突破关键位**: 突破近期高点/低点
5. **多周期共振**: 多个时间框架同时显示同方向
6. **振幅巨大**: 24h内价格区间跨度大

## 🔍 策略1: 基础版

### 适用场景
- 快速扫描全市场
- 不需要K线数据,扫描速度快
- 适合初筛,发现潜在机会

### 评分体系 (满分100分)

| 维度 | 权重 | 说明 |
|------|------|------|
| 24h涨跌幅 | 30分 | 涨跌幅越大,得分越高 |
| 价格位置 | 20分 | 当前价相对24h区间的位置 |
| 成交量 | 20分 | 24h成交量绝对值 |
| 波动率 | 15分 | 24h振幅大小 |
| 趋势持续性 | 15分 | 价格偏离中间价的程度 |

### 评分标准

#### 1. 24h涨跌幅 (30分)
- ≥50%: 30分 (极强)
- ≥30%: 25分 (很强)
- ≥20%: 20分 (强)
- ≥10%: 15分 (中等)
- ≥5%: 10分 (弱)

#### 2. 价格位置 (20分)
做多时:
- 价格在24h区间90%以上: 20分 (极强势)
- 75%-90%: 15分 (强势)
- 60%-75%: 10分 (中等)

#### 3. 成交量 (20分)
- ≥5000万USDT: 10分 (巨额)
- ≥1000万USDT: 8分 (大量)
- ≥100万USDT: 5分 (中等)

#### 4. 波动率 (15分)
- ≥50%: 15分 (剧烈波动)
- ≥30%: 12分 (大幅波动)
- ≥20%: 8分 (中等波动)

#### 5. 趋势持续性 (15分)
- 偏离度≥80%: 15分 (强单边)
- 偏离度≥60%: 12分 (较单边)
- 偏离度≥40%: 8分 (中等)

### 评级系统

- **🔥 极强机会** (≥90分): 立即关注!
- **🔥 强机会** (≥75分): 重点关注
- **🔥 中等机会** (≥60分): 可以关注
- **⚠️ 弱机会** (≥40分): 谨慎观察
- **❌ 无机会** (<40分): 忽略

## 🚀 策略2: 高级版

### 适用场景
- 深度分析,精准捕捉
- 需要多时间框架K线数据
- 扫描速度较慢,但准确度高

### 核心检测模块

#### 1. 短期爆发力检测 (35分) ⭐⭐⭐⭐⭐

**5分钟K线分析**:
- 最近5根K线累计涨幅 ≥2%: 得分
- 连续同向K线数 ≥3根: 额外加分

**15分钟K线分析**:
- 最近5根K线累计涨幅 ≥5%: 得分

**设计理由**: 单边趋势的起点往往伴随短期的快速拉升,5分钟和15分钟K线能最早捕捉到爆发信号。

#### 2. 成交量放大确认 (25分) ⭐⭐⭐⭐

**5分钟量比**:
- 当前成交量 / 前5根平均 ≥3倍: 最高15分

**15分钟量比**:
- 当前成交量 / 前5根平均 ≥2.5倍: 最高10分

**24h成交量**:
- ≥5000万USDT: +5分
- ≥1000万USDT: +3分

**设计理由**: 量价齐升是趋势确认的关键信号,成交量放大说明有大资金介入。

#### 3. 突破关键位 (20分) ⭐⭐⭐⭐

检查是否突破近N周期高点/低点:
- 突破20周期高点: 20分
- 接近突破(95%位置): 10分

**设计理由**: 突破意味着趋势加速,是追涨的最佳时机。

#### 4. 多周期共振 (15分) ⭐⭐⭐

检查1H、4H时间框架是否与主方向一致:
- 完全共振(所有周期同向): 15分
- 部分共振: 按比例给分

**设计理由**: 多周期共振说明不是短期反弹,而是真正的趋势。

#### 5. 24h趋势强度 (5分)

- ≥50%: 5分
- ≥30%: 4分
- ≥20%: 3分

## 🔧 如何集成到扫描器

### 修改 engine.py 支持K线数据

在 `engine.py` 的 `scan_once()` 方法中,为高级版策略添加K线数据获取:

```python
# 如果是高级版单边趋势扫描器
if hasattr(self.strategy, '_analyze_unilateral_trend_advanced'):
    # 为每个symbol获取多周期K线
    for inst_id in symbols:
        ticker = self.get_ticker_data(inst_id)
        symbol = self.parse_ticker(ticker)

        # 获取多周期K线
        symbol.extra_data['klines'] = {
            '5m': self.get_klines(inst_id, bar='5m', limit=50),
            '15m': self.get_klines(inst_id, bar='15m', limit=50),
            '1h': self.get_klines(inst_id, bar='1H', limit=50),
            '4h': self.get_klines(inst_id, bar='4H', limit=50),
        }

        all_symbols_data.append(symbol)
```
