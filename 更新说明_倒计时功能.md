# 自动扫描倒计时功能更新说明

## 更新内容

### 1. 新增倒计时显示
- 在扫描信息区域添加了倒计时标签
- 显示格式：`⏱ 下次扫描倒计时: MM:SS`
- 倒计时颜色根据剩余时间变化：
  - 蓝色（> 30秒）：正常状态
  - 黄色（10-30秒）：即将扫描
  - 红色（< 10秒）： imminent 扫描

### 2. 自动扫描逻辑优化
- **倒计时定时器**：每秒更新一次倒计时显示
- **自动触发扫描**：倒计时结束后自动开始扫描
- **扫描完成后重置**：扫描完成后自动重置倒计时
- **结果显示和入库**：扫描结果自动显示并添加到交易对池

### 3. 用户体验改进
- 倒计时期间可以随时停止自动扫描
- 扫描期间倒计时暂停显示
- 扫描完成后自动开始下一轮倒计时

## 使用方法

1. **启动自动扫描**：
   - 选择一个扫描策略
   - 设置扫描间隔（1-60分钟）
   - 点击"▶ 自动扫描"按钮

2. **查看倒计时**：
   - 在"实时扫描信息"区域查看倒计时
   - 倒计时颜色提示剩余时间

3. **停止自动扫描**：
   - 点击"⏹ 停止自动"按钮

## 技术实现

### 修改的文件
- `src/ui/scanner_page.py`

### 新增的方法
- `_update_countdown()`: 更新倒计时显示
- 修改 `_auto_scan_tick()`: 处理倒计时和自动扫描触发
- 修改 `toggle_auto_scan()`: 初始化倒计时
- 修改 `on_scan_finished_internal()`: 扫描完成后重置倒计时

### 新增的UI组件
- `self.countdown_label`: 倒计时显示标签

## 工作流程

```
启动自动扫描
    ↓
初始化倒计时（扫描间隔 × 60 秒）
    ↓
每秒更新倒计时显示
    ↓
倒计时 = 0？
    ├─ 否 → 继续倒计时
    └─ 是 → 开始扫描
              ↓
         显示扫描进度
              ↓
         扫描完成
              ↓
         显示结果并入库
              ↓
         重置倒计时
              ↓
         继续下一轮...
```

## 注意事项

- 扫描期间倒计时暂停
- 手动停止扫描会同时停止倒计时
- 扫描间隔设置修改后需要重新启动自动扫描才能生效
